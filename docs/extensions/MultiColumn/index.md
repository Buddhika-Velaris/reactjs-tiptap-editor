# Multi Column

The MultiColumn extension allows you to create multi-column layouts in your editor.

## Usage

```tsx
import { ColumnActionButton } from 'reactjs-tiptap-editor/multicolumn'; // [!code ++]

const extensions = [
  ...,
  // Import Extensions Here
  ColumnActionButton,  // [!code ++]
];

```
