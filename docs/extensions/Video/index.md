# Video

The Video extension allows you to add a video to your editor.

## Usage

```tsx
import { Video } from 'reactjs-tiptap-editor'; // [!code ++]

const extensions = [
  ...,
  // Import Extensions Here
  Video // [!code ++]
];
```
